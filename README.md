# ASM_FIRST_PROJ
Implement an assembly language program that reads from the command line a number, doubles that number, and then prints out "The double is: " and the doubled number.

# HOW TO USE
This program will accept any double or integer input and will double and print that input given the result is within the max double range. To compile on windows: gcc .\doubleinput.s -o exenamehere, ./exenamehere to run. After running, enter an integer or a double, and the program will print the doubled value.
